# The IdeaPad ACPI Extras kernel modules for ThinkBook 2024 G6+

This kernel module solves problem with laptop turning off after closing the lid.

Tested on archlinux and Ubuntu 24.04 6.9 kernel.

## Build

Just run:

```shell
make
```

## Install

Run

```shell
make install
```

## Install via DKMS

```shell
make install-dkms
```
